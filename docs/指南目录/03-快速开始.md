# 快速开始

本章节将帮助您快速搭建欧贝项目开发环境。

## 环境要求

在开始之前，请确保您的开发环境满足以下要求：

- **Node.js**: 版本 18.0 或更高
- **包管理器**: npm、yarn 或 pnpm

::: tip 推荐版本
推荐使用 Node.js LTS 版本（如 20.x）以获得最佳兼容性。
:::

## 安装

### 创建项目

```bash
# 克隆项目仓库
git clone https://github.com/your-org/obei.git

# 进入项目目录
cd obei

# 安装依赖
npm install
```

### 启动开发服务器

```bash
npm run dev
```

启动成功后，访问 `http://localhost:5173` 即可预览项目。

## 项目结构

```
obei/
├── docs/               # 文档目录
│   ├── .vitepress/     # VitePress 配置
│   ├── guide/          # 指南文档
│   └── api/            # API 文档
├── src/                # 源代码目录
├── public/             # 静态资源目录
└── package.json        # 项目配置
```
